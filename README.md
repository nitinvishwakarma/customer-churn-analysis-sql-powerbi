# Customer Churn Analysis | SQL & Power BI

## Project Overview
This project focuses on identifying key attrition indicators and assessing churn determinants using a combination of advanced SQL data transformation and Power BI visualization.

## Key Technical Features
* **Data Extraction & Transformation:** Utilized SQL CTEs and Stored Procedures to process complex customer datasets for analysis.
* **Feature Analysis:** Analyzed customer demographics, service usage, and account data to identify primary drivers of churn.
* **Actionable Dashboards:** Developed interactive Power BI reports that enable stakeholders to implement data-driven retention strategies.
* **Business Impact:** Bridged the gap between raw database records and strategic business outcomes through clear KPI visualization.

## Tools Used
* **SQL Server:** T-SQL, Common Table Expressions (CTEs), Stored Procedures.
* **Power BI Desktop:** Interactive reporting and data modeling.
* **Data Cleaning:** Power Query and SQL-based preprocessing.
